# backend-playground

A collection of backend experiments and proof-of-concepts across different languages and frameworks.

## Projects

### Java
- `java/descartes-pit` - Mutation testing experiments with PIT
- `java/person-record` - Java records API exploration
- `java/graphql` - GraphQL server with Java
- `java/jazzer` - Fuzz testing with Jazzer
- `java/string-templates` - Java string templates (JEP 430)

### Zig
- `bitonic-generator/bitonic-api` - Bitonic sequence generator REST API built with Zig and facil.io, includes k6 load tests and Docker support

### Node.js
- `node/lifecycle-events` - Node.js process lifecycle events
- `node/next-tick` - process.nextTick and event loop experiments
